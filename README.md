📊 Supply Chain & Inventory Optimization Dashboard

📌 Project Overview This project analyzes supply chain and inventory data using Power BI to improve operational efficiency, reduce costs, and enable smarter decision-making. The dashboard provides a comprehensive view of inventory levels, demand patterns, supplier performance, and order trends.

🎯 Objectives

Track real-time inventory levels
Identify overstock and stockout risks
Optimize reorder points
Evaluate supplier performance
Analyze demand trends for better forecasting
🛠 Tools & Technologies

Power BI (Data Visualization)
CSV Dataset (Data Source)
DAX (Data Analysis Expressions)
Data Modeling
📂 Dataset Description

The dataset includes:

Product ID and category
Inventory levels
Order quantities
Supplier details
Lead time
Demand data
📈 Dashboard Features

📦 Inventory Status (Low / Optimal / Excess)
📉 Demand vs Supply Analysis
🚚 Supplier Performance Tracking
⏳ Lead Time Analysis
📊 Trend Analysis (Time-based insights)
📊 Key KPIs

Inventory Turnover Ratio
Stockout Rate
Carrying Cost of Inventory
Order Fulfillment Rate
Average Lead Time
Demand Forecast Accuracy
🚀 Insights Generated

Identified products with frequent stock shortages
Detected excess inventory leading to higher holding costs
Highlighted suppliers with delayed deliveries
Suggested optimized reorder levels
📌 Conclusion

The dashboard helps organizations streamline supply chain operations, reduce inefficiencies, and improve inventory management through data-driven insights.
